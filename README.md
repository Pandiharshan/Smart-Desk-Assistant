# Smart Desk Assistant

An intelligent workspace management solution that combines IoT and AI to create the perfect work environment. Control your desk lighting, manage work/break cycles, and get AI-powered productivity insights.

![Smart Desk Assistant Demo](https://via.placeholder.com/800x400.png?text=Smart+Desk+Assistant+Demo)

## üåü Features

- **Smart Lighting Control**: Seamlessly control your desk lighting with a beautiful Flutter interface
- **AI-Powered Productivity**: Get intelligent work/break cycle recommendations
- **Real-time Monitoring**: Monitor your desk environment in real-time
- **Cross-Platform**: Works on Android, iOS, and web (coming soon)
- **NodeMCU Integration**: Connect with your NodeMCU device for hardware control

## ü§ñ AI Integration

Our Smart Desk Assistant leverages AI to enhance your productivity:

- **Adaptive Work Cycles**: AI analyzes your work patterns to suggest optimal work/break intervals
- **Smart Lighting Adjustment**: Automatically adjusts lighting based on time of day and ambient conditions
- **Productivity Insights**: Get weekly reports on your work patterns and suggestions for improvement

### Current AI Features

- Simple ambient light adjustment

## üöÄ Getting Started

### Prerequisites
- Flutter SDK (latest stable version)
- Dart SDK (included with Flutter)
- NodeMCU ESP8266 (for hardware integration)
- Android Studio / Xcode (for mobile development)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Pandiharshan/Smart-Desk-Assistant.git
   cd Smart-Desk-Assistant
   ```

2. Install dependencies:
   ```bash
   flutter pub get
   ```

3. Run the app:
   ```bash
   flutter run
   ```

## üîß Hardware Setup

1. Flash your NodeMCU with the provided firmware
2. Connect your desk lighting to the NodeMCU
3. Power the NodeMCU and ensure it's on the same network as your mobile device
